Підсумки:

Завдання 1. Алгоритм реалізований у функції "find_min" має складність O(h),
де h — висота дерева.

Завдання 2. Необхідно обійти всі вузли дерева, тому складність алгоритму становить O(n),
де n — кількість вузлів у дереві.

Завадння 3. Відомо, що операція heapify - O(n).
Далі використовуються функції:

- heappop - O(log n)
- heappush - O(log n)

Кожен елемент бере участь у таких операціях, тому загальна складність алгоритму становить - O(n log n)
